# LocalProofHQ Homepage

A clean, modern, one-page homepage for LocalProofHQ, designed to build trust and encourage small businesses to use our platform to collect and showcase social proof, customer feedback, and reputation signals.

## Features
- Clean, professional, and mobile-responsive design
- Fast and modern React + Vite stack
- Brand colors: Deep Blue (#003366), Light Blue (#66CCFF)
- Modern sans-serif fonts (Inter, Lato, Open Sans)

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Run the development server:
   ```sh
   npm run dev
   ```

## Project Structure
- `src/` - Main source code
- `public/` - Static assets
